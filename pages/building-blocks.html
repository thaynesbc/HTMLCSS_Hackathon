Building Blocks:
  • there are a few HTML elements that are used specifically to apply CSS styling. the following css will be used to demonstrate:
    .example-class {
    color: #ff0000;
    font-weight:bold;
    text-decoration:underline;
    border: 1px solid orange;
}
    • span: the span element is applied inline. it may be useful for things like creating tooltips or formatting important text. eg:
        this is outside of the span element and this is inside and this is outside.
        this is outside of the span element and <span class="example-class">this is inside</span> and this is outside.
    • div: the div (divider or section) element is applied as a block. it is useful for creating banners, headers, and cards. eg:
        this is outside of the div element and
        this is inside
        and this is outside.
        this is outside of the div element and <div class="example-class">this is inside</div> and this is outside.
    • section: the section element behaves the same as the div element. it is typically used where content is being meaningfully grouped based on a common theme; ie. section is used to indicate how content (text, images, apps) are organized, while div is more generic and geared toward the layout and styling of a website.
  • divs are the building blocks of websites. they determine where content is positioned, how it is sized, borders, backgrounds, etc.
  • divs can be nested. the "outer" div acts as a parent, while "inner" divs are the children. children will inherit properties from the parent unless told otherwise. for example:
    
<div class="parent">
    <div class="child1">text</div>
    <div class="child2">text</div>
</div>
