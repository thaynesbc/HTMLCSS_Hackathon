CSS Rules:
  • CSS is used to style HTML. A .css file is also called a 'stylesheet'.
  • as with HTML, CSS is written in lowercase.
  • structure: CSS is made up of a selector and declarations. declarations are separated by a semicolon ; and are all contained within curly brackets { } :
            header {
            font-size: 34px;
            font-weight:bold;
            background-color: gold;
        }
  • within each declaration is a property and value:
            header {
            font-size: 34px;
            font-weight: bold;
            background-color: gold;
        }
  • <header>this is a header!</header> creates...
    this is a header!
  • there are different types of selector:
    • elements: existing HTML elements, for example: <p>, <b>, <h1>... 
                h1 {
            color: red;
            font-size: 32px;
            font-weight: bold;
        }
        will cause all text in the <h1>hello!</h1> element to show as

        hello!

    • id: used for a user-defined element that is unique within a page. 
                #example2 {
            color: green;
            text-decoration:underline;
        }
        will cause the <p id="example2">hello!</p> element to show as

        hello!

    • class: used for a user-defined element that may be used multiple times across a site or page. 
                .example3 {
            color: magenta;
            border-bottom: 3px solid orange;
            width: 50%;
        }
        will cause the <p class="example3">hello!</p> element to show as

        hello!

  • more information on CSS selectors can be found here
  • CSS can be added to HTML in three ways:
    • internal: CSS is written directly in the HTML file using the <style></style> element.
      for hub sites, this is the best practice, particularly if CSS is written into the HTML header; it keeps all of the CSS in one place for easy editing.
    • external: a stylesheet is imported into the HTML using
      <link rel="stylesheet" href="mystyle.css">
      for a self-hosted website, eg. GitHub Pages, this is best practice.
    • inline: CSS is referenced inline using a style attribute:
      <div style="property:value;"></div>
      this should be used sparingly, as it can be difficult to come back to for later edits and updates.
