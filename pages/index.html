HTML Rules:
  • structure: HTML elements are denoted by '<' and '>' brackets, eg:
    <b>the 'b' element makes text bold</b>
  • it is best practice to close all elements.
    • opening an element looks like this: <element>
    • closing an element looks like this: </element>.
    • some elements, like <br>, do not close. in this case, it is best practice to write <br />
  • close elements in the same order you opened them, eg:
    correct: <body><b><u>hello world</u></b></body>
    incorrect: <body><b><u>hello world</b></u></body>
  • elements should be lowercase, ie. <body> instead of <BODY>
  • elements can have attributes. attribute values are attached to attributes by the equal sign ('=') and must be in quotation marks, eg:
    <img src="value" width="value">
    note: you should not have spaces between the attribute, equal sign, and value
  • more detailed standards and examples can be found here.
  • information on escape characters (ie. how to display 'special' characters in html) can be found here
